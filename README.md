# Создать текстовый файл с ёлкой следующего вида (TypeScript + Playwright):
                    W

                    *

               @* * * * *

             * * * * * * * * *@

        @ * * * * * * * * * * * * *

     * * * * * * * * * * * * * * * * *@

                  TTTTT

                  TTTTT

## Входные параметры:
- количество уровней ёлки,
- путь к выходному файлу.
- 
**Результат:** txt-файл с сгенерированной елкой.
  
**Бонус:** написать приемочные тесты для этой программы.

**Бонус со звездочкой:** настроить GitHub Actions для автозапуска приемочных тестов на каждый Pull Request.

**UPD:** Приёмочные тесты на проверку обработки ошибок не стал писать.

### ПО для работы с проектом

Перед началом работы убедитесь, что на вашем устройстве установлены следующие ПО:
1. IntelliJ IDEA или другой инструмент для работы.
2. GIT
3. Браузер - Google Chrome
4. Node.js
### Установка и запуск

Склонировать репозиторий на свой рабочий стол
```
git@github.com:DmitriyRonMan/christmasTreeGenerator.git
```
Открыть проект в вашем инструменте

Для запуска генерации ёлки использовать команду (6 - кол-во уровней (1-6), tree.tx - сохраняемый файл с ёелкой (можно менять))
```
npx ts-node treeGenerator.ts 6 tree.tx 
```

Для запуска тестов использовать команду
```
npx playwright test  
```
